<analysis>**original_problem_statement:**
The user wants to build a full-stack pharmacy delivery service application named RX Expresss, which should replicate the functionalities of the DrugLift app. The only exception is that all routing, delivery management, and proof-of-delivery functionalities will be handled by integrating a third-party service, Circuit Spoke.

Recent user requests have focused on enhancing the admin dashboard with advanced order management, optimization, and real-time tracking capabilities.

PRODUCT REQUIREMENTS:
- **Core Goal:** Replicate DrugLift's feature set.
- **Key Features:**
  - Multi-role authentication (Admin, Pharmacy, Driver, Patient).
  - Pharmacy Portal: Dashboard, creating/managing deliveries (with pricing visibility and QR code generation), QR code scanning.
  - Driver Portal: View and manage assigned deliveries, scan packages, capture Proof of Delivery, mark copay as collected.
  - Admin Dashboard: An advanced, interactive dashboard for full operational control. This includes a Smart Organizer for grouping orders by area/time, drag-and-drop route assignment, route optimization previews with live maps, and real-time driver tracking.
  - Patient Experience: Real-time public tracking page.
- **Mandatory Integrations:** Circuit Spoke, Google Maps, Stripe, Twilio, SendGrid.
- **Admin Controls:**
    - Full control over user accounts (drivers, pharmacies).
    - Ability to change the status of any delivery at any time.
    - Control over pricing for delivery types and add-ons.
    - Advanced route planning and driver management tools.
- **QR Code System:** Borough-prefixed QR codes (Q, B, M, S, X) generated for each order.
- **Proof of Delivery (POD):** Drivers must be able to capture photo and digital signatures as proof of delivery.

**User's preferred language**: English

**what currently exists?**
A full-stack application with a React frontend and a FastAPI backend connected to MongoDB.
- **Backend:** Supports multi-role authentication and a comprehensive set of API endpoints for managing users, orders, pharmacies, drivers, pricing, and QR codes. It includes sophisticated endpoints for route optimization (location-based), order reassignment, and fetching live driver locations.
- **Frontend:** Features a login page with role-based access.
    - **Admin Dashboard:** A highly advanced dashboard featuring:
        - Interactive stat cards and a Ready for Pickup section.
        - **Orders Management** with a Smart Organizer view that groups orders by borough and time window.
        - **Drag-and-Drop** functionality to re-assign orders between time windows.
        - **Quick Driver Assignment** via a dropdown on each order card.
        - **Route Optimization Preview** in a modal, showing an optimized sequence based on geographic proximity.
        - **Live Map Visualization** within the preview, displaying the route path and real-time driver locations.
    - **Pharmacy Portal:** Allows creating deliveries with copay amounts and viewing real-time pricing.
    - **Copay Feature:** Fully implemented and tested, allowing pharmacies to add copay amounts and admins to track collection status.
